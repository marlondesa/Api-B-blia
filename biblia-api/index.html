Bíblia API — Almeida Corrigida Fiel (ACF)

API estática gratuita com os 66 livros da Bíblia completa.

Endpoints

GET /api/index.json           → Todos os livros
GET /api/{livro}/index.json   → Info do livro
GET /api/{livro}/{cap}.json   → Versículos do capítulo

Exemplos

  • api/index.json — Lista todos os livros
  • api/joao/3.json — João capítulo 3
  • api/salmos/23.json — Salmos 23
  • api/genesis/1.json — Gênesis 1

Total

66 livros · 31.106 versículos · Versão ACF
